자연어 입력만으로 데이터센터 마이크로그리드의 최적 경제급전을 도출하는

## 슬라이드 7 — ② 다이캐스팅 공정 이상탐지 (XGBoost·SHAP) (2024)
`[축: ML 모델링]`  **★ ML 모델 개발 + 평가지표 + 지표(파생변수) 설계 + XAI 한 장에 다 들어감**
```
[목표]
다이캐스팅 공정의 시계열 패턴으로 불량을 예측하고, SHAP로 불량 원인까지 해석

[역할]
파생변수 설계 · 모델링 · 모델 해석 (제4회 K-인공지능 제조데이터 분석 경진대회)

[문제 상황]
· 공정 변수의 시계열 패턴이 일정하지 않아 불량 원인 파악이 어려움
· 불량 클래스 비율이 매우 낮은 심한 불균형 데이터
· 경험적 관리에 의존 → 무엇이 불량을 만드는지 정량 설명이 필요

[문제 접근·해결]
· 변수를 현재상태·변동성·변화추세·계절성·시간대 5가지 관점으로 분해
· 롤링 윈도우(window=5)로 이동평균·변동계수(CV) 파생변수 생성, 금형코드·가열로 단위로 그룹 차별 계산
· T-검정으로 파생변수 유의성(p<0.05) 검증, CCF로 시차(lag)별 영향 확인
· 불균형 보정(클래스 가중치) + XGBoost → Accuracy 0.997 / Recall 0.950 / Precision 0.979 / F1 0.965
· SHAP로 상·하부 금형 온도의 현재상태·변동성이 핵심 불량 인자임을 규명

· 가설(파생변수)을 통계검정으로 검증하고 ML로 구현하는 흐름은, flex의 'EDA·가설수립·지표 설계·ML 모델 개발' 과정과 동일
· 심한 불균형 데이터에서 Precision·Recall·F1로 모델을 평가·튜닝한 경험으로, 퇴사·번아웃 같은 희귀 사건 예측 모델의 지표 설계·평가에 기여

## 슬라이드 8 — ③ 난임 시술 임신 성공 예측 (LG Aimers 6기) (2024)
`[축: ML 모델링]`  **★ '평가지표가 바뀌면 모델 전략을 바꾼다' — flex의 지표 이해도를 정확히 보여줌**
```
[목표]
난임 IVF 시술 데이터로 임신 성공을 예측 (예선=분류 / 본선=확률회귀)

[역할]
파생변수 설계 · 평가지표 맞춤 모델링 · 앙상블/스태킹 (팀 프로젝트, 최종 7위)

[문제 상황]
· 예선과 본선의 예측 대상·평가 방식이 완전히 다름
  - 예선: 성공 여부(0/1), ROC-AUC
  - 본선: 성공 확률(0~1), Weighted Brier + F1 혼합 Score
· 임신 성공(희귀 클래스) 비율이 낮음, 범위형 문자열("1-5회",">20") 다수

[문제 접근·해결]
· 의미 기반 파생변수: 배아 이식·ICSI 비율, 난자 수정 성공률, 연령 가중치 등 + 상관분석으로 핵심 변수 선별
· 예선: CatBoost(Optuna 튜닝) + AutoGluon 가중평균 앙상블 → 0.70 → 0.74, 본선 진출
· 본선: 평가식 가중치 wᵢ를 손실함수에 직접 반영해 희귀 클래스에 민감하게 학습
        CatBoost+LightGBM Stacking으로 예측 오류 영역 보완 → 0.64 → 0.66

[flex 기여]
